## ❓ これは何ができるの？

**MudaRemote** は **Mudae bot（ムダエ・ボット）** です。DiscordでMudaeを **あなたの代わりに自動で** 遊んでくれるプログラムです。

できること：

- 🎲 **Auto roll Mudae** — `$wa` や `$ha` などのロールを自動で送ります。
- 💍 **Mudae auto claim** — 良いキャラが出たら、すぐに自動で捕まえます。
- 💎 **Mudae auto kakera** — クリスタル（カケラ）を自動でクリックしてお金を貯めます。
- 🎯 **Discord Mudae sniper** — 他の人が出した欲しいキャラを「横取り（スナイプ）」します。
- 🤖 **Mudae slash commands bot** — `/wa` などのスラッシュコマンドを使ってカケラを10%多くもらえます。
- 👥 **Mudae multi-account sync** — サブ垢と一緒に協力して動けます。
- 🛡️ **Anti-ban stealth** — DiscordにBANされないように、人間のような自然な動きをします。
- 🖥️ **簡単な操作画面** — 難しいコードは不要。ボタンをクリックするだけで設定できます。

> **⚠️ 注意事項:** これは「セルフボット」です。セルフボットは **Discordのルール違反** です。アカウントが **BANされる可能性があります**。このボットは **学習目的** のものです。使う場合は **自己責任** でお願いします。私たちは責任を負いません。

## ✨ 機能の一覧

### 🎯 キャラを捕まえる — Mudae Auto Claim

自分や他人のガチャを監視して、一番良いキャラを捕まえます。

| 機能 | 内容 |
| :--- | :--- |
| **ウィッシュリスト捕獲** | 欲しいキャラをリストにしておけば、出た瞬間に捕まえます。 |
| **シリーズ捕獲** | 「ナルト」が好きなら、ナルトの全キャラを自動で狙います。 |
| **高額キャラ狙い** | 高価なキャラ（例：カケラ500以上）を他人から奪います。 |
| **即時キャッチ** | 自分が回している時、良いのが出たらその瞬間に捕まえます。 |
| **パニック・キャッチ** | 時間切れになりそうな時、無駄にしないよう何でも捕まえます。 |
| **無料カード捕獲** | クリスマス等の無料イベントキャラを自動で手に入れます。 |
| **自動 $rt** | `$rt` を使って、残りの捕獲回数を自動で回復します。 |
| **事後 $rt** | キャラを捕まえた後、すぐに `$rt` をして次を備えます。 |
| **ブラックリスト** | いらないキャラを除外できます。ボットは無視し続けます。 |

---

### 💎 カケラ — Mudae Auto Kakera

カケラ（色のついたボタン）を賢くクリックしてお金稼ぎをします。

| 機能 | 内容 |
| :--- | :--- |
| **自動クリック** | 自分や他人のロールに出たカケラを自動で押します。 |
| **優先順位** | 複数のボタンがある時、一番価値が高いものから押します。 |
| **パワー管理** | 体力をチェックして、パワーが足りない時は無理に押しません。 |
| **自動 $dk** | パワーが減ったら、自動で `$dk` をして回復します。 |
| **カオス・モード** | 消費パワーが50%少ない「カオス・カケラ」を優先的に狙えます。 |
| **MK限定モード** | `$mk` ロールのカケラだけを狙い、パワーを節約します。 |
| **スフィア検知** | 消費パワー0の「スフィア」は見逃さず必ず押します。 |
| **カスタム制限** | 「パワーが80%以上の時だけクリックする」といったルールが作れます。 |
| **重複防止** | 同じものを2回クリックしてパワーを無駄にすることはありません。 |

---

### 🎲 ロール（ガチャ） — Auto Roll Mudae

一番賢いタイミングで自動でガチャを回します。

| 機能 | 内容 |
| :--- | :--- |
| **自動ロール** | `$wa` などを自動で送信し続けます。 |
| **スラッシュコマンド** | `/wa` を使い、BAN耐性を上げつつカケラを多く稼ぎます。 |
| **賢いタイミング** | 自分のキャッチ（結婚）回数がリセットされる「直後」にガチャが終わるように計算します。 |
| **予約実行** | 「毎日 14:00 と 18:30 に回す」といった設定ができます。 |
| **自動 $us** | 貯まっているロール（$us）を自動で使い切ります。 |
| **自動 $mk** | カケラ専用ロール（$mk）を使い、出てきたカケラを捕ります。 |
| **自動 $rolls** | ロールが切れたら、自動でおまけロール（$rolls）をもらいます。 |
| **ボーナス検知** | カケラをクリックして得た「おまけロール」を即座に使います。 |
| **おこぼれ作戦（ルーカー）** | 他の人が回している時は盗み（スナ）に専念し、最後に一気に自分も回します。 |
| **キー稼ぎ** | 捕獲ができなくても、キーを稼ぐために回し続けます。 |

---

### 🛡️ ステルス（隠密）設定 — BAN対策 / ゴーストモード

人間らしく振る舞い、ボットであることを隠します。

| 機能 | 内容 |
| :--- | :--- |
| **ランダムな待ち時間** | リセット後にすぐ動かず、0〜40分の間でランダムに待ってから始めます。 |
| **会話検知** | チャンネルで誰かが話している間は空気を読んで静かに待機します。 |
| **ランダムな絵文字** | キャラを捕まえる時のハート絵文字を毎回バラバラに変えます。 |
| **クリックの遅延** | カケラを押す前に 0.3〜1.0秒ほど「人間らしい」間を置きます。 |
| **おやすみタイマー** | 「午前1時から7時までは寝る」といった設定ができます。完全に停止します。 |
| **メンテナンス検知** | Mudaeがメンテ中の時は止まり、終わってから自然に再開します。 |
| **スラッシュ固定** | スラッシュコマンドを使う設定の時、絶対にテキストコマンドを送りません。 |

---

### 👥 マルチアカウント同期 — Mudae Multi-Account Sync

複数のアカウントを繋げて、効率よく遊べます。

| 機能 | 内容 |
| :--- | :--- |
| **メイン垢同期** | サブ垢が「メイン垢のウィッシュ（Wished by @Main）」を見つけたら、即座に捕まえます。 |
| **個別プロファイル** | アカウントごとに別のチャンネル、リストを設定できます。 |
| **同時実行** | ボタン一つで、登録した全アカウントを同時に動かせます。 |
| **自動復旧** | もし止まってしまっても、60秒後に自動で再開します。 |

### 🖥️ 簡単な設定ツール (GUI)

コードをいじる必要はありません。`mudae_preset_editor.py` を開けば：
- ✅ トークンとチャンネルIDを記入
- ✅ チェックボックスで機能をON/OFF
- ✅ ウィッシュリストを追加
- ✅ カケラのルールを決定
- ✅ Windows起動時に自動で始める設定

これらすべてがマウス操作で完結します。

## 🛠️ インストール方法

### 必要なもの
- **[Python 3.8 以降](https://www.python.org/downloads/)** — インストール時に ✅ **"Add to PATH"** にチェックを入れてください。
- Discordトークン（[取得方法はこちら](#-discordトークンの取り方)）

### 手順1: ボットをダウンロード
GitHubからZIPをダウンロードして解凍するか、以下のコマンドを使います：
```bash
git clone https://github.com/misutesu-desu/MudaRemote.git
cd MudaRemote
```

### 手順2: 必要なソフトをインストール
フォルダの中でコマンドプロンプトを開き、以下を入力します：
```bash
pip install discord.py-self inquirer requests
```

### 手順3: 設定を開く
```bash
python mudae_preset_editor.py
```
**トークン**、**チャンネルID** を入力して **💾 Save Changes** を押します。

### 手順4: ボット開始
設定画面の **▶ Launch Bot** をクリックします。これで完了です。🎉
