## ❓ 이게 무엇인가요?

**MudaRemote**는 **Mudae 봇**입니다. 디스코드에서 Mudae 게임을 **당신을 대신해서 자동으로** 플레이해주는 프로그램입니다.

주요 기능:

- 🎲 **Auto roll Mudae** — 자동으로 주사위 명령(`$wa`, `$ha` 등)을 보냅니다.
- 💍 **Mudae auto claim** — 좋은 캐릭터가 나오면 즉시 자동으로 잡습니다(결혼).
- 💎 **Mudae auto kakera** — 자동으로 카케라를 클릭해서 돈을 모읍니다.
- 🎯 **Discord Mudae sniper** — 다른 사람이 뽑은 캐릭터를 가로챕니다(스나이핑).
- 🤖 **Mudae slash commands bot** — 슬래시 명령(`/wa`)을 사용하여 카케라를 10% 더 받습니다.
- 👥 **Mudae multi-account sync** — 부계정들과 함께 협력해서 플레이할 수 있습니다.
- 🛡️ **Anti-ban stealth** — 정지당하지 않도록 사람과 똑같이 행동합니다.
- 🖥️ **쉬운 GUI** — 코드 수정 없이 버튼 클릭만으로 모든 설정을 마칠 수 있습니다.

> **⚠️ 주의사항:** 이 프로그램은 "셀프 봇"입니다. 셀프 봇 사용은 **디스코드 약관 위반**입니다. 계정이 **영구 정지**될 위험이 있습니다. 이 오픈소스는 **학습용**입니다. 사용 시 발생하는 모든 책임은 **본인**에게 있습니다.

## ✨ 기능 리스트 (쉬운 설명)

### 🎯 캐릭터 잡기 — Mudae Auto Claim

자신과 타인의 롤을 감시하고 최고의 캐릭터를 잡습니다.

| 기능 | 설명 |
| :--- | :--- |
| **위시리스트 점유** | 원하는 캐릭터 목록을 만들면, 나타나는 순간 즉시 잡습니다. |
| **시리즈 점유** | 예: "나루토"를 설정하면 나루토의 모든 캐릭터를 자동으로 잡습니다. |
| **가치 점유** | 일정 금액(예: 500 카케라) 이상의 비싼 캐릭터를 남들로부터 가로챕니다. |
| **즉시 본인 점유** | 내가 돌리는 도중 좋은 캐릭터가 나오면 즉시 잡습니다. |
| **패닉 점유** | 다음 리셋까지 시간이 얼마 안 남았다면 아무 캐릭터나 잡아서 횟수를 낭비하지 않습니다. |
| **이벤트 카드 점유** | 크리스마스나 새해 무료 이벤트 카드를 자동으로 모읍니다. |
| **자동 $rt** | `$rt`를 사용해서 캐릭터를 더 잡을 수 있게 자동으로 횟수를 회복합니다. |
| **블랙리스트** | 절대로 잡고 싶지 않은 캐릭터를 설정해서 무시할 수 있습니다. |

---

### 💎 카케라 자동화 — Mudae Auto Kakera

카케라(색깔 버튼)를 똑똑하게 클릭해서 돈을 벌어줍니다.

| 기능 | 설명 |
| :--- | :--- |
| **자동 클릭** | 나 또는 다른 사람들의 롤에 나온 카케라를 자동으로 누릅니다. |
| **우선순위 설정** | 버튼이 여러 개일 때 가장 비싼 것을 먼저 누릅니다. |
| **에너지 관리** | 남은 에너지를 확인해서 에너지가 부족할 때는 클릭하지 않습니다. |
| **자동 $dk** | 에너지가 부족하면 자동으로 `$dk`를 사용해 충전합니다. |
| **카오스 모드** | 에너지가 50% 적게 드는 카오스 카케라만 골라서 누를 수 있습니다. |
| **MK 전용 모드** | `$mk` 롤에 나온 카케라만 클릭해서 에너지를 아낍니다. |
| **스피어 감지** | 에너지가 들지 않는 스피어는 항상 클릭합니다. |
| **중복 클릭 방지** | 이미 누른 것을 다시 누르지 않아 에너지를 낭비하지 않습니다. |

---

### 🎲 자동 주사위 — Auto Roll Mudae

가장 적절한 타이밍에 주사위를 굴립니다.

| 기능 | 설명 |
| :--- | :--- |
| **자동 롤** | 자동으로 `$wa`, `$ha`, `$ma` 등을 보냅니다. |
| **슬래시 명령** | `/wa`를 사용하여 카케라 10% 보너스를 받고 정지 위험을 낮춥니다. |
| **스마트 타이밍** | 캐릭터 잡기 횟수가 충전되는 시간에 맞춰 롤이 끝나도록 시간을 계산합니다. |
| **예약 반복** | "매일 14:00 와 18:30 에 돌려라"와 같은 설정이 가능합니다. |
| **자동 $mk** | 카케라 전용 롤($mk)을 사용하고 나타나는 카케라를 누릅니다. |
| **루커 모드** | 다른 사람들이 주사위를 굴릴 때까지 기다렸다가 남들이 뽑은 좋은 캐릭을 훔친 뒤, 마지막에 본인의 주사위를 다 씁니다. |

---

### 🛡️ 보안 — Anti-Ban / 고스트 모드

MudaRemote가 안전한 이유입니다. 다른 봇들은 기계처럼 행동하지만, 이 봇은 사람처럼 행동합니다.

| 기능 | 설명 |
| :--- | :--- |
| **랜덤 지연** | 리셋 후 즉시 실행하지 않고 0~40분 사이의 무작위 시간을 기다립니다. |
| **채팅 감지** | 채널에서 사람들이 대화 중이면 조용해질 때까지 기다립니다. |
| **무작위 반응** | 캐릭터를 잡을 때 사용하는 하트 이모지를 매번 다르게 사용합니다. |
| **클릭 지연** | 카케라를 누를 때 0.3~1.0초 사이의 사람 같은 지연 시간을 둡니다. |
| **취침 모드** | "오전 1시부터 7시까지는 잠자기"를 설정하면 그 시간에는 완전히 멈춥니다. |
| **점검 감지** | Mudae가 점검 중이면 자동으로 멈추고 끝날 때까지 기다립니다. |

## 🛠️ 설치 방법 (단계별 안내)

### 준비물
- **[Python 3.8 이상](https://www.python.org/downloads/)** — 설치 시 ✅ **"Add to PATH"** 체크 필수
- 디스코드 토큰 ([아래 확인](#-디스코드-토큰-따는-법))

### 1단계: 봇 다운로드
GitHub에서 ZIP을 받거나 아래 명령어를 입력하세요:
```bash
git clone https://github.com/misutesu-desu/MudaRemote.git
cd MudaRemote
```

### 2단계: 필수 프로그램 설치
폴더 안에서 터미널(CMD)을 열고 입력하세요:
```bash
pip install discord.py-self inquirer requests
```

### 3단계: 설정 화면 열기
```bash
python mudae_preset_editor.py
```
**토큰**과 **채널 ID**를 넣고 **💾 Save Changes**를 누르세요.

### 4단계: 시작
설정 화면에서 **▶ Launch Bot**을 누르면 실행됩니다! 🎉
